# Reporting And Analytics

## Overview

The platform provides reporting and analytics at three levels:

1. Super Admin
2. Organization Admin
3. Member

Analytics must always respect role-based permissions.

Users must only see data they are authorized to access.

---

# Analytics Hierarchy
